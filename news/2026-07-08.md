# ข่าวประจำวันที่ 2026-07-08

## AI Model & Product Updates

- Anthropic ขยาย **Claude Cowork** จากแอปเดสก์ท็อปให้ใช้งานผ่านเว็บและมือถือได้แล้ว (สำหรับผู้ใช้แผน Max) เริ่มงานจากคอมพิวเตอร์ที่ออฟฟิศ เช็คสถานะจากมือถือ และรับผลลัพธ์ได้แม้ปิดเครื่องแล้ว
- จากสถิติการใช้งาน Cowork พบว่างานที่ถูกใช้มากที่สุดคือ "งานประมวลผลธุรกิจ" (ดึงข้อมูลกระจัดกระจายมาทำรายงาน, ทำเช็คลิสต์ onboarding, กระทบยอดสเปรดชีต) สัดส่วน 33.4% ตามด้วยงานเขียนคอนเทนต์/สื่อสาร 16.4%
- OpenAI ทยอยปล่อยโมเดลเสียง/มัลติโมดัลรุ่นใหม่ **gpt-realtime-2.1** และ **gpt-realtime-2.1-mini** ลด latency ระดับ p95 ได้อย่างน้อย 25% พร้อมปรับปรุงการจดจำเสียงและการใช้เครื่องมือ (tool use)

## US–China AI War

- สำนักข่าว Reuters เปิดเผยว่า **DeepSeek** สตาร์ทอัพจีน กำลังพัฒนาชิป AI สำหรับงาน inference ของตัวเอง เพื่อลดการพึ่งพาชิปจาก Nvidia และ Huawei โดยเน้นขั้นตอนที่โมเดลตอบคำถามผู้ใช้ (ไม่ใช่การเทรนโมเดล) ซึ่งเป็นส่วนที่ใช้พลังงานน้อยกว่าและต้นทุนต่ำกว่า GPU ทั่วไป
- Huawei Ascend 910C ยังแข่งขันกับ GPU ระดับกลางของ Nvidia ได้ แต่คอขวดสำคัญคือหน่วยความจำ HBM ซึ่งผู้ผลิตในจีนอย่าง CXMT ยังผลิตได้ไม่ถึงมาตรฐานเชิงพาณิชย์สำหรับงานเทรน AI ขนาดใหญ่
- กระทรวงพาณิชย์สหรัฐฯ ยืนยันมาตรการควบคุมส่งออกชิป AI ครอบคลุมถึงบริษัทลูกของจีนที่ตั้งอยู่นอกประเทศจีนด้วย เพื่อปิดช่องโหว่การอ้อมซื้อชิปผ่านบริษัทในเครือ

## AI Deals & Promotions

- วันนี้เป็นวันแรกที่ **Claude Fable 5** ต้องใช้ "usage credits" สำหรับผู้ใช้ทุกระดับแพ็กเกจ (Pro/Max/Team/Enterprise) ในอัตรา API มาตรฐาน **10 ดอลลาร์สหรัฐ/ล้านโทเคนอินพุต และ 50 ดอลลาร์สหรัฐ/ล้านโทเคนเอาต์พุต** ซึ่งแพงกว่า Claude Opus 4.8 (5/25 ดอลลาร์) ถึง 2 เท่า หลังโปรโมชันรวมในแพ็กเกจ (สูงสุด 50% ของโควตารายสัปดาห์) สิ้นสุดลงเมื่อวานนี้
- ผู้ใช้แผน Consumer ที่ต้องการซื้อเครดิตเพิ่มเพื่อใช้ Fable 5 เกินโควตาฟรี 50% ต่อสัปดาห์ ต้องยืนยันตัวตนผ่านระบบ Persona ก่อน
- ราคาโปรโมชันของ **Claude Sonnet 5** ในฐานะโมเดลเริ่มต้นของ Claude Code ยังคงอยู่ที่ 2/10 ดอลลาร์สหรัฐต่อล้านโทเคน (อินพุต/เอาต์พุต) ต่อเนื่องถึง 31 สิงหาคม 2569
- คู่แข่งจากจีนอย่าง Z.ai คิดราคา ZCode ที่ 1.40/4.40 ดอลลาร์สหรัฐต่อล้านโทเคน ถูกกว่า Claude Code อย่างมีนัยสำคัญ กดดันให้ผู้ใช้งานฝั่งองค์กรเริ่มเปรียบเทียบต้นทุนมากขึ้น

## AI Adoption & Use Cases

- Microsoft เปิดตัวหน่วยธุรกิจใหม่ **Microsoft Frontier Company** ทุ่มงบลงทุน 2.5 พันล้านดอลลาร์สหรัฐ และดึงผู้เชี่ยวชาญ 6,000 คน เข้าไปช่วยลูกค้าองค์กรออกแบบ-ติดตั้ง-ปรับปรุงระบบ AI ให้เกิดผลลัพธ์ทางธุรกิจจริง หลัง Microsoft, OpenAI และ Anthropic รวมกันทุ่มงบไปแล้วกว่า 8 พันล้านดอลลาร์สหรัฐ เพื่อแก้ปัญหาการ deploy AI ในองค์กรที่ยังติดขัด
- งานวิจัยของ MIT Project NANDA พบว่าแม้องค์กรทั่วโลกลงทุนด้าน Generative AI ไปแล้ว 30-40 พันล้านดอลลาร์สหรัฐ แต่ถึง 95% ขององค์กรยังไม่เห็นผลตอบแทนที่วัดผลได้ชัดเจน มีเพียงราว 5% ของโปรเจกต์ทดลอง (pilot) เท่านั้นที่สร้างกำไรจริงในงบดุล
- ในไทย เครือ **CP All (7-Eleven)** เดินหน้าปั้นบุคลากรด้าน AI ทั้งภายในและภายนอกองค์กร ตั้งเป้าให้พนักงานกว่า 60% ใช้งาน AI ในการทำงานภายใน 5 ปีข้างหน้า (ถึงปี 2571)
- กรณีศึกษาจาก HP ที่นำโมเดลของ OpenAI มาใช้ในทีมวิศวกร สามารถประมวลผล pull request ได้ 122 รายการ ครอบคลุม 43 โปรเจกต์ภายในไม่กี่สัปดาห์ และแก้ไขช่องโหว่ความปลอดภัยซอฟต์แวร์ได้ภายในวันเดียว จากเดิมที่คาดว่าต้องใช้เวลาถึง 1 เดือน
